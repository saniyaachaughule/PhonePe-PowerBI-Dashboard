# PhonePe End-to-End Data Analysis Dashboard

## Project Overview

This project is an interactive Power BI dashboard built using a PhonePe transaction dataset containing over **150,000 transaction records**. The dashboard provides insights into various PhonePe services, including Insurance, Loans, Money Transfer, and Recharge & Bills.

The objective of this project is to analyze transaction trends, monitor payment success and failure rates, identify failed payment reasons, and visualize service-wise performance through an intuitive dashboard.


## Project Objectives

* Analyze transaction data across multiple PhonePe services.
* Monitor successful and failed payment transactions.
* Identify major reasons for payment failures.
* Visualize monthly transaction trends.
* Compare service-wise transaction performance.
* Build an interactive dashboard for business decision-making.
